A config-driven, ultra-performant Astro + React Three Fiber portfolio architecture. Designed for elite engineers requiring both an immersive WebGL experience and a zero-JS "Recruiter Mode" fallback for Applicant Tracking Systems (ATS).

## 🧪 Test-Driven Development (TDD)
This project rigorously adheres to TDD principles. We write tests before implementation, ensuring 100% test coverage across all React islands, state management (Zustand), and utility logic.
- **Framework:** Vitest + React Testing Library
- **Goal:** 100% Coverage (Statements, Branches, Functions, Lines)

To run the test suite:
```bash
npm run test
```
